INTERNET

Invention: Originated from ARPANET in 1969, with TCP/IP developed in the 1970s.

Evaluation: Evolved from academic use to a global communication tool.

Working: Utilizes protocols like TCP/IP to transmit data packets across networks.

____________________________________________________________________________________________________________________________________________________

BROWSER

Invention: The first graphical browser, Mosaic, was introduced in 1993.

Evaluation: Changed how users interact with the web by rendering HTML and multimedia.

Working: Retrieves and renders data from web servers using HTTP/HTTPS protocols.

____________________________________________________________________________________________________________________________________________________

APPLICATION

Invention: Evolved from early computer programs to multi-functional software.

Evaluation: Includes desktop, mobile, and web-based apps for productivity and communication.

Working: Executes tasks via interaction with the operating system and hardware.

____________________________________________________________________________________________________________________________________________________

WEB APPLICATION TECHNOLOGY

Invention: Advanced with AJAX and APIs, enabling real-time interactions.

Evaluation: Central to modern ecosystems, offering platform-independent services.

Working: Follows a client-server model where the browser requests data from a server.
